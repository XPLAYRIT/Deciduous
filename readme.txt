== Description ==

Deciduous is a highly extensible WordPress Theme featuring 5 widget areas, drop-down menus, several layout options, and a whole lot more. It is fully responsive meaning it will adapt the layout to smaller screens seamlessly. Perfect for any blog and an awesome starting point for child theme development.

Features:

* Perfect to use as-is or as a blank WordPress theme for development
* Search-Engine Optimized and compatible with popular SEO plugins
* 5 widget-ready areas and more possible in your Child Theme
* Internationalized into many languages
* Responsive stylesheet makes the layout adapt to smaller screens
* Several layout options available in the Customizer
* Editable footer text to remove the theme credit in the theme customizer
* Options for multi-author blogs
